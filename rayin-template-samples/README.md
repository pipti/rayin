# 行业样例
共享行业模板样例，方便行业应用。

## 文件名规范
data.json : 数据，固定文件名。
data_desc.json : 数据说明，固定文件名。
element1.html : 构件，可自定义名称。
tpl.json : 模板，固定文件名。
preview_n.png：预览图，n为序号。
字体文件单独放入fonts中

## 运行
导入项目后直接运行 PdfBoxGeneratorOpenhtmltopdfSamples测试类。 \
测试程序默认查找所有tpl.json，对应目录下的data.json \
生成目录至tmp目录下
